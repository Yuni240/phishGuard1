# PhishGuard Phishing-Mail Detection Tool

A lightweight, browser-based tool designed to analyze email content and highlight common phishing indicators.

## Features
- Detects suspicious keywords and patterns
- Clean and simple HTML interface
- Instant analysis in the browser
- No external dependencies
- Suitable for training and awareness demos

## How to Run
1. Clone the repo: git clone https://github.com/Yuni240/phishGuard1.git
2. Open the file: xdg-open phishing_mail.html

Runs entirely in the browser. No installation required.

## Live Hosting (GitHub Pages)
Enable via: Settings → Pages → Branch: main → /root

- HTML5
